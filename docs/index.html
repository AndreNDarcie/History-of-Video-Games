Definite Best Games of All Time

The project uses web crawler and scraping techniques to aggregate multiple lists of influential sources and combines everything into one public list

Fork Star Watch

About:

  • All data is extracted automatically by scripts written in Python.

Rules:

  • The game has to be mentioned in at least three different sources
  • The source must be among the 500,000 most popular sites according to the Alexa Page Rank
  • The position in which the game appears matters more than the number of mentions
  • Games are ordered by their final score

Final Score:

  • The final score is generated from the position in which the game appears in the multiple lists.
  • For example, position 1 equals 99 score points and position 2 equals 98 score porints and so on consecutively.
  • Takes (position where the game appears) minus 100 and turn the result value into positive
  • The final score is the sum of all scores
